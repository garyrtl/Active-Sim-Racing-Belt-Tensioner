# Active-Sim-Racing-Belt-Tensioner
Active Sim Racing Belt Tensioner

https://youtu.be/jDv2dkZlcOc

Material:

1 x ODESCv3.6 56V https://sequremall.com/collections/odesc/products/odesc-dual-drive

1 x 24V 20A power supply

2 x Hoverboard motors

2 x SK16 Linear Shaft Support. https://www.amazon.com/PATIKIL-Aluminum-Linear-Clamping-Support/dp/B0DC5WTQNY

2 x Bicycle Brake Cable

Some M4, M5, M6, M8 bolts and nuts

Instruction:
1. https://www.tinkercad.com/things/5T1yV7gz4mY-active-belt-tensioner. Export pulley spool as STL. Print it with 100% infill.
2. https://docs.odriverobotics.com/v/0.5.6/hoverboard.html. Set up ODESC for hoverboard motors.
3. https://github.com/garyrtl/Active-Sim-Racing-Belt-Tensioner/blob/main/belt-tensioner-settings.shsds
Import belt-tensioner-settings.shsds into SimHub Custom Serial Device.
4. Select ODrive 3.6 CDC in Serial port.
